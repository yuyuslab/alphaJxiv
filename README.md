# 主な機能（06/08/2025 現在）
1. 現状は Chrome 拡張機能である
2. [Jxiv](https://jxiv.jst.go.jp/index.php/jxiv)のPDFの隣にサイドバーを設ける
3. Google アカウント認証後に Jxiv で PDF を表示するページにてコメントができる
